## 一、机器强制型 —— Hook 自动注入（唯一会主动"跳出来"的提示）

| 项 | 内容 |
|---|---|
| 配置文件 | `C:\Users\EDY\.claude\settings.json` → `hooks.PreToolUse` |
| 事件 | `PreToolUse` |
| matcher | `Write\|Edit` |
| 性质 | 软提醒，不拦截、不弹窗、不打断 |
| 生效范围 | 全局，所有项目 |
| 触发时机 | 每次 `Write` / `Edit` 工具执行**前** |

**注入文案（原文）：**

> 提醒(全局CLAUDE.md 1):改代码前先给出【根因+修改大纲+单行检查清单 [备份|根因|大纲|已确认]】并等用户确认，再动手 Edit/Write。

**hook 配置片段：**

```json
"hooks": {
  "PreToolUse": [
    {
      "matcher": "Write|Edit",
      "hooks": [
        {
          "type": "command",
          "command": "echo '{\"hookSpecificOutput\":{\"hookEventName\":\"PreToolUse\",\"additionalContext\":\"提醒(全局CLAUDE.md 1):改代码前先给出【根因+修改大纲+单行检查清单 [备份|根因|大纲|已确认]】并等用户确认，再动手 Edit/Write。\"}}'"
        }
      ]
    }
  ]
}
```
